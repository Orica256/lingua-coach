# 開発ロードマップ

プログラミング初心者でも進めやすいよう、フェーズごとに分割。各フェーズ完了時点で動くものができることを目指す。

## Phase 0: 環境構築 (1週間)

### ゴール
ローカルで「Hello World」が表示されるNext.jsアプリが動く状態。

### タスク
- [ ] Node.js 20.x LTS をインストール
- [ ] Git をインストール、GitHub アカウント作成
- [ ] VSCode + Claude Code拡張をセットアップ
- [ ] GitHub にPrivateリポジトリ作成
- [ ] `npx create-next-app@latest` でプロジェクト初期化
  - TypeScript: Yes / Tailwind: Yes / App Router: Yes / ESLint: Yes
- [ ] shadcn/ui 初期化 (`npx shadcn-ui@latest init`)
- [ ] Anthropicアカウント作成、APIキー発行（予算上限 $20 を設定）
- [ ] Supabaseアカウント作成、プロジェクト作成
- [ ] Vercelアカウント作成、GitHubリポジトリと連携
- [ ] `.env.local` セットアップ、`.gitignore` 確認
- [ ] gitleaks インストール、pre-commitフック設定

### 完了条件
- `npm run dev` でローカル起動
- Vercelに自動デプロイされる
- リポジトリにAPIキーが含まれていないことをgitleaksで確認

## 進め方のコツ（初心者向け）

1. **完璧を目指さない**: 各フェーズ「動く」まで来たら次へ
2. **Claude Codeを最大活用**: わからない箇所はClaude Codeに「このエラーの意味は?」「このコードを説明して」と聞く
3. **小さくコミット**: 機能単位で頻繁にgit commit
4. **Vercelで毎日デプロイ**: 自動で本番反映されるので、自分でも毎日触る
5. **セキュリティは後回しにしない**: Phase 1 から最低限のRLS・APIキー管理は徹底
